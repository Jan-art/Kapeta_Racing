 3) Using "1" Waving Animation from Mixamo (https://www.mixamo.com/#/?page=1&query=waving)

 4)  Using "1" (TROPHY-CUP) icon from Flaticon (<a href="https://www.flaticon.com/free-icons/cup" title="cup icons">Cup icons created by Freepik - Flaticon</a>)

 5) Using Sound sources from ZapSlat (Royalty Free Sounds) (https://www.zapsplat.com/sound-effect-category/royalty-free-music/)

 6) All fonts used are standar Unity fonts
